# Gtk-hello

Bored are you? 

Our Application's Button has a solution for it ;)

## Dependencies

```
sudo apt install git valac meson

```

## Install

```
git clone https://github.com/forhooman/gtk-hello.git
```
```
cd gtk-hello
```
```
meson build --prefix=/usr
```
```
cd build && ninja
```
```
ninja install
```
